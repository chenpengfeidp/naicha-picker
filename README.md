# 今天喝什么奶茶？🧋

> 献给每一个站在奶茶店门口纠结 10 分钟的你。

## 为什么做这个？

每次约朋友、约对象出去喝奶茶，最痛苦的不是排队，而是——**选哪家**。

喜茶？CoCo？蜜雪冰城？古茗？一点点？17 个品牌在脑子里打架，最后往往以"随便吧"收场。

**现在，把选择权交给命运。**

## 功能

- 🎰 **3D 旋转轮盘** — 17 大热门奶茶品牌在 3D 环形轮盘上旋转展示
- 🎯 **一键抽奖** — 点击按钮，轮盘高速旋转后缓缓停下，命运之茶浮出水面
- 🎊 **彩纸庆祝** — 抽中品牌弹窗展示，配合全屏彩纸撒花特效
- 📱 **移动端适配** — 专为手机设计，分享链接即可使用
